---
description: Save current changes to git (add, commit, push)
---

1. Run `git status` to view the current state of the repository.
2. Run `git add .` to stage all changes.
3. Generate a concise but descriptive commit message based on the changes.
4. Run `git commit -m "Your generated message"`.
5. Run `git push` to upload the changes to the remote repository.
